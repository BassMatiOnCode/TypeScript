Functions

Functions are annotated as follows:

function function-name( annotated-parameter-list ) : return-type-name { statements-block }

The annotated parameter list is a comma-separated list of parameters, each with a type annotation:

parameter-name : type-name

Anonymous functions are often closely tied to a calling context, and therfore parameters and return types can usually be inferred:

var names : string[] ;
names.forEach( function( s ) { console.log( s.toUpperCase( )); } ) ;

It is obvious that forEach expects a function that takes an array element of type string and returns nothing, so you don't have to annotate anything. The annotated version looks like this:

function( s : string ) : void { ... }

Similarly, arrow functions are annotated like this:

( annotated-parameter-list ) : return-type-name => { statements }

Example:

let a = ["john", "joe", "james"];
console.log( a );
a = a.map( s => s[0].toUpperCase() + s.substr(1) );
a = a.map( (s : string) : string => s[0].toUpperCase() + s.substr(1));
console.log( a );

The map method takes a function expression and calls this function with the members of the array a. The function expression declares an arrow function which takes a string parameter s and returns a string type.

Here with explicit type annotations:

a = a.map( (s : string) : string => s[0].toUpperCase() + s.substr(1));
